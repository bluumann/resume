JAVASCRIPT CHEAT SHEET

Useful Javascript keywords and methods

let   keyword

The let keyword was introduced in ES6 (2015). Variables defined with let cannot be Redeclared. Variables defined with let must be Declared before use. Variables defined with let have Block Scope.

  • Before ES6 (2015), JavaScript had only Global Scope and Function Scope.

  • ES6 introduced two important new JavaScript keywords: let and const.
  • These two keywords provide Block Scope in JavaScript.
  • Variables declared inside a { } block cannot be accessed from outside the block

const   keyword

The const keyword was introduced in ES6 (2015). Variables defined with const cannot be Redeclared. Variables defined with const cannot be Reassigned. Variables defined with const have Block Scope.
When to use JavaScript const? As a general rule, always declare a variable with const unless you know that the value will change. Use const when you declare:

  • A new Array
  • A new Object
  • A new Function
  • A new RegExp

slice ( )   method

slice() extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters: the start position, and the end position (end not included). Remember: JavaScript counts positions from zero. First position is 0.
If a parameter is negative, the position is counted from the end of the string.

Math.random ( )   method

Math.random() returns a random number between 0 (inclusive), and 1 (exclusive) Math.random() always returns a number lower than 1.
